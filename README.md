# 👥 KeenKeeper — Keep Your Friendships Alive

KeenKeeper is a personal relationship management tool designed to help you
maintain and nurture your most important connections. In a busy world, it's easy
to lose track of when you last spoke to a friend. KeenKeeper ensures those
meaningful connections never fade away.

## 🚀 Features

- **Friendship Dashboard**: A bird's-eye view of all your connections with
  status indicators (On Track, Almost Due, Overdue).
- **Interactive Timeline**: Automatically log calls, texts, and video chats to
  keep a history of your interactions.
- **Friendship Analytics**: Visualize your interaction patterns with beautiful
  charts to see how you're spending your time.
- **Smart Reminders**: Track days since last contact and set personal goals for
  how often you want to reach out.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop
  viewing.

## 🛠️ Technologies Used

- **React.js**: Frontend library for building the user interface.
- **React Router DOM**: For seamless page navigation and deep linking.
- **Tailwind CSS**: Utility-first CSS framework for rapid and consistent
  styling.
- **shadcn/ui**: High-quality accessible components for a polished look.
- **Recharts**: Data visualization library for friendship analytics.
- **Lucide React**: Beautifully simple icons.
- **Motion**: For smooth animations and transitions.
- **React Hot Toast**: Elegant toast notifications for user feedback.
